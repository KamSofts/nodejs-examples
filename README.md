# nodejs-examples
SERVER
```bash
mkdir server
cd server/
npm init -y
npm install dotenv nodemailer
```
CLIENT
```bash
mkdir client
npm create vite@latest .
    React 
    Javascript
```

Server-side send email example :-
```bash
cd server/
node send_email/index.js
```

Client-side cricket score board :-
```bash
cd client/
npm run dev
```
